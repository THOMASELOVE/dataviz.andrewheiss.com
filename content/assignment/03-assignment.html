Task 0: Understand Markdown

  • Go through the seven short lessons of this Markdown tutorial.This tutorial has you use _s for italics, like _this_. Markdown also accepts single * for italics, like *this*. I prefer writing with asterisks for both *italic* and **bold**, but underscores work just fine too—it’s a matter of personal preference. Similarly, the tutorial uses asterisks for lists. Markdown will also accept dashes.

  • Read through this RMarkdown tutorial to see how you embed chunks of R code in a Markdown document and knit that document into different output formats like HTML, Word, and PDF.There are lots of R Markdown files in the wild, like R for Data Science and even this website.
